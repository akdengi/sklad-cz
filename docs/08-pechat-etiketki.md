# 8. Печать этикетки товара

## Что печатается

### Товары с маркировкой (ЧЗ)

Этикетка содержит:
- **DataMatrix** — код маркировки ЧЗ (КМ)
- **EAN-13** — штрихкод товара
- **Название товара**
- **Артикул**
- **Дата**

### Товары без маркировки

Этикетка содержит:
- **Название товара**
- **Артикул**
- **Дата**
- **EAN-13** — штрихкод товара
- **Без** DataMatrix кода маркировки

## Размеры этикеток

| Размер | Описание |
|--------|----------|
| 50x50 мм | Квадратная (по умолчанию) |
| 58x40 мм | Альбомная |
| 50x60 мм | Вертикальная |
| 58x30 мм | Альбомная, узкая |
| 50x25 мм | Маленькая |

## Варианты макета

### Для товаров с маркировкой

| Макет | Содержимое |
|-------|-----------|
| **Полный** | DataMatrix + название + артикул + дата + EAN-13 |
| **Только КМ** | Только DataMatrix код |
| **Только штрихкод** | DataMatrix + EAN-13 |

### Для товаров без маркировки

| Макет | Содержимое |
|-------|-----------|
| **Полный** | Название + артикул + дата + EAN-13 |
| **Только КМ** | Название + EAN-13 (DataMatrix не генерируется) |
| **Только штрихкод** | Название + EAN-13 |

## Печать через интерфейс

1. Перейдите во вкладку **«Этикетки»**
2. Выберите товар (SKU)
3. Выберите размер этикетки
4. Выберите макет
5. Укажите номера единиц (например, `#1-#5` или `#1,#3,#7`)
6. Укажите количество копий
7. Выберите формат печати:

### Формат PDF
- Нажмите **«PDF»**
- Скачается PDF-файл с этикетками
- Печатайте на любом принтере

### Формат ZPL (Zebra)
- Нажмите **«ZPL»**
- Скачается файл `.zprn`
- Отправьте на принтер Zebra через драйвер или raw-печать

### Формат TSPL (TSC)
- Нажмите **«TSPL»**
- Скачается файл `.tspl`
- Отправьте на принтер TSC

### Прямая печать
- Нажмите **«Печать»** — отправит команды напрямую на принтер (если настроен)

## Генерация КМ для этикеток

Для товаров с маркировкой, если у единицы ещё нет реального КМ:
1. Система генерирует серийный номер на основе GTIN-14 и номера единицы
2. Используется детерминированный алгоритм (SHA-256)
3. Один и тот же SKU + номер единицы всегда дают одинаковый код

Если у единицы уже есть реальный КМ — используется он.

Для товаров без маркировки КМ не генерируется.

## Поиск по КМ

Если у вас есть КМ и нужно найти соответствующий товар:
1. Введите КМ в поле поиска на вкладке «Этикетки»
2. Система найдёт SKU и выберет его автоматически
