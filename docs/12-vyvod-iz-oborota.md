# 12. Вывод товара из оборота

## Что это такое

Вывод из оборота — это процесс снятия товара с учёта в системе Честный Знак. Выполняется после продажи товара конечному покупателю.

## Когда подавать отчёт

- После продажи товара (через кассу или маркетплейс)
- В течение **3 рабочих дней** с момента продажи
- На странице «Сводка» отображаются дедлайны

## Порядок действий

### Шаг 1: Оформите продажу в приложении

1. Продайте товар через вкладку **«Продажа»**
2. Убедитесь, что статус изменился на **«Продан»**
3. Заполнены все данные: номер заказа, цена, адрес

### Шаг 2: Подайте отчёт через ЛК ЧЗ

1. Войдите в личный кабинет Честный Знак: https://markirovka.crpt.ru
2. Перейдите в раздел **«Отчёты»** → **«Отчёт о выводе из оборота»**
3. Создайте новый отчёт
4. Заполните:
   - **ИНН** организации
   - **Код продукции** (GTIN)
   - **Серийный номер** — из КМ кода
   - **Причина вывода**: «Дистанционная продажа» / «Продажа» / другое
   - **Дата вывода**
   - **Документ-основание**: номер чека или заказа
5. Подпишите отчёт ЭЦП
6. Отправьте

### Шаг 3: Обновите статус в приложении

1. Через 1-2 дня перейдите в **«Настройка»**
2. Нажмите **«Обновить статусы всех товаров»**
3. Статус товара должен измениться на **RETIRED** (Выбыл) или **WITHDRAWN** (Выведен)
4. Статус отчёта автоматически станет **«Подтверждено ЧЗ»**

## Экспорт данных для отчёта

Для удобства подачи отчёта:

1. Перейдите в **«Настройка»**
2. Нажмите **«Экспорт выбытия (CSV)»**
3. В файле будут:
   - Полный КМ код
   - Код для отчёта (всё до GS+91)
   - Формат для маркетплейса (с `\u001d`)

## Отслеживание отчётов

Во вкладке **«Вывод из оборота»** отображаются:
- Единицы, требующие отчёта (статус **«Не начато»**)
- Подтверждённые отчёты (статус **«Подтверждено ЧЗ»**)

### Изменение статуса отчёта

Статус отчёта обновляется **автоматически** при проверке статуса ЧЗ:

| Статус ЧЗ | Статус отчёта | Описание |
|-----------|---------------|----------|
| INTRODUCED | **Не начато** | Товар в обороте, отчёт не подан |
| RETIRED / WITHDRAWN / WRITTEN_OFF | **Подтверждено ЧЗ** | Товар выбыл, отчёт подтверждён |

> Ручное изменение статуса отчёта на «Подтверждено ЧЗ» доступно только если статус ЧЗ = RETIRED/WITHDRAWN/WRITTEN_OFF.

### Колонки таблицы

| Колонка | Описание |
|---------|----------|
| **Статус отчёта** | Не начато / Подтверждено ЧЗ |
| **Статус в ЧЗ** | Текущий статус в системе Честный Знак |

## Массовая проверка отчётов

1. Отметьте нужные товары **чекбоксами** во вкладке «Вывод из оборота»
2. Нажмите **«Проверить статус в ЧЗ (N)»**
3. Система проверит каждый товар и обновит статусы

## Возврат товара

Если товар был возвращён покупателем, нажмите кнопку **«Вернуть в оборот»** напротив товара со статусом «Подтверждено ЧЗ». Подробнее: [Возврат товара](13-vozvrat.md).

## Типы выбытия

| Тип | Описание |
|-----|----------|
| **Перед отгрузкой** | Товар отгружается покупателю |
| **При возврате товара** | Товар возвращён от покупателя |

## Причины выбытия

| Причина | Когда использовать |
|---------|-------------------|
| Дистанционная продажа | Продажа через интернет-магазин / маркетплейс |
| Возврат при дистанционном способе продажи | Возврат интернет-заказа |
| Прочее | Другие случаи |

## Важно

- Отчёт нужно подавать в течение 3 рабочих дней
- Без подачи отчёта товар останется в статусе «В обороте»
- Статус RETIRED в ЧЗ = товар выбыл, дальнейшие операции невозможны
