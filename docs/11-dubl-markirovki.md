# 11. Создание дубля маркировки (этикетки) для упаковки

## Когда нужен дубль

- Оригинальная этикетка повреждена
- Нужно разместить маркировку на внешней упаковке
- Требуется дополнительная этикетка для логистики

## Способ 1: Печать дубля через вкладку «Этикетки»

1. Перейдите во вкладку **«Этикетки»**
2. Выберите товар (SKU)
3. В поле **«Номера единиц»** укажите номер нужной единицы (например, `#3`)
4. Выберите размер и макет
5. Укажите **количество копий: 2** (или больше)
6. Нажмите **«PDF»** или **«Печать»**

### Пример

Если у товара «Футболка» 10 единиц и нужен дубль для 3-й:
- Номера единиц: `#3`
- Количество копий: 2
- Получите 2 одинаковых этикетки с одним КМ

## Способ 2: Печать этикетки для упаковки (multipack)

Если нужно разместить маркировку на коробке с несколькими единицами:

1. Используйте **макет «Только КМ»** — он содержит только DataMatrix код
2. Размер: **50x25 мм** (компактный)
3. Напечатайте на самоклеящейся бумаге
4. Наклейте на упаковку

## Способ 3: Генерация из приложения

Если оригинальная этикетка утеряна и нужно восстановить КМ:

1. Перейдите во вкладку **«Остатки»**
2. Найдите единицу товара
3. Нажмите **«Изменить»**
4. Скопируйте **Код маркировки ЧЗ**
5. Перейдите во вкладку **«Этикетки»**
6. Вставьте скопированный КМ в поле поиска
7. Система найдёт SKU и предложит печать

## Форматы для different принтеров

| Формат | Когда использовать |
|--------|-------------------|
| **PDF** | Любой принтер, включая домашний |
| **ZPL** | Принтеры Zebra (производственные) |
| **TSPL** | Принтеры TSC |

## Важно

- Дубль этикетки содержит **тот же КМ**, что и оригинал
- Один КМ = одна единица товара
- Дублирование этикетки не создаёт новую единицу в базе
- Для печати дублей используйте термобумагу для долговечности
