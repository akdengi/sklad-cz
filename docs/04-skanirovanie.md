# 4. Занесение КМ маркировки сканированием

## Что такое КМ маркировки

Код маркировки (КМ) — это DataMatrix код, содержащий:
- **01** — GTIN-14 (14 цифр)
- **21** — серийный номер
- **91** — ключ аутентификации (необязательно)
- **92** — криптохвост (необязательно)

Пример структуры: `010460000000000021СЕРИЙНЫЙНОМЕР`

## Способ 1: Сканирование одной единицы

### Через вкладку «Сканирование»

1. Перейдите во вкладку **«Сканирование»**
2. Выберите:
   - **Товар (SKU)** — какой товар маркируется
   - **Склад** — на каком складе физически находится товар
   - **Статус** — статус присваиваемый единице (обычно «Нанесён»)
3. Наведите сканер на штрихкод товара или вставьте код вручную
4. Счётчик увеличится на 1
5. Повторяйте для каждой единицы

### Что происходит при сканировании

1. Система ищет первую непримаркированную единицу выбранного SKU на складе
2. Если находит — присваивает ей отсканированный код
3. Если не находит — создаёт новую единицу с этим кодом
4. Дубли отклоняются (код уже существует в базе)

### Через вкладку «Остатки»

1. Перейдите во вкладку **«Остатки»**
2. Найдите единицу без КМ (выделена жёлтым)
3. Нажмите **«Изменить»**
4. Вставьте КМ в поле «Код маркировки ЧЗ»
5. Сохраните

## Способ 2: Массовое сканирование

### Через вкладку «Сканирование»

1. Перейдите во вкладку **«Сканирование»**
2. Настройте параметры (SKU, склад, статус)
3. Сканируйте коды последовательно
4. Каждый код обрабатывается автоматически
5. История сканирования отображается справа
6. Используйте **«Сбросить счётчик»** для новой партии

### Полезные кнопки

- **«Обработать»** — обработать вставленный текст (несколько кодов)
- **«Очистить историю»** — очистить список обработанных кодов
- **«Сбросить счётчик»** — обнулить счётчик сканирований

## Способ 3: Быстрое создание единиц

Если нужно заранее создать пустые ячейки для последующего сканирования:

1. Перейдите во вкладку **«Товары»**
2. Нажмите **«Завод единиц»**
3. Выберите SKU и склад
4. Укажите количество
5. Нажмите **«Создать»**

После этого сканируйте коды — они будут привязываться к созданным ячейкам.

## Формат кода

Код маркировки должен содержать минимум 18 символов и начинаться с `01` (GTIN). Система автоматически:
- Удаляет символы FNC1 (`\xe8`) и GS (`\x1d`)
- Обрезает код до AI 91 (проверочные данные)
- Удаляет пробелы и лишние символы
