# 14. Вывод товара из оборота вне продажи

## Когда это нужно

- Товар списан (брак, порча, истечение срока годности)
- Товар передан бесплатно (дарение, образцы)
- Товар утерян
- Экспорт за пределы ЕАЭС
- Выбытие по другим причинам, не связанным с продажей

## Порядок действий

### Шаг 1: Оформите выбытие в приложении

1. Перейдите во вкладку **«Остатки»**
2. Найдите нужный товар
3. Нажмите **«Изменить»**
4. В секции **«Выбытие»** (жёлтый фон) заполните:
   - **Тип операции**: «Перед отгрузкой»
   - **Причина выбытия** (см. таблицу ниже)
   - **Тип документа**: прочее, товарная накладная, акт приёма-передачи, кассовый чек, УПД
   - **Наименование документа**: название документа-основания
   - **Номер документа**
   - **Дата документа**
   - **Цена за единицу** (руб)
   - **Адрес выбытия**: адрес места выбытия
   - **FIAS ID**: UUID адреса из ФИАС
5. Сохраните

### Шаг 2: Подайте отчёт через ЛК ЧЗ

1. Войдите в личный кабинет Честный Знак: https://markirovka.crpt.ru
2. Перейдите в раздел **«Отчёты»** → **«Отчёт о выводе из оборота»**
3. Создайте новый отчёт
4. Заполните:
   - **ИНН** организации
   - **Код продукции** (GTIN)
   - **Серийный номер** — из КМ кода
   - **Причина вывода** (см. таблицу)
   - **Дата вывода**
   - **Документ-основание**: реквизиты документа
5. Подпишите отчёт ЭЦП
6. Отправьте

### Шаг 3: Обновите статус отчёта

1. После отправки отчёта измените статус на **«Отправлено в ЧЗ»**
2. После подтверждения ЧЗ статус изменится на **«Подтверждено ЧЗ»**

## Причины выбытия вне продажи

| Причина | Когда использовать |
|---------|-------------------|
| **Списание** | Брак, порча, истечение срока |
| **Безвозмездная передача** | Дарение, образцы, тестирование |
| **Потеря/хищение** | Утерян или украден |
| **Экспорт за пределы ЕАЭС** | Вывоз из Евразийского союза |
| **Производство** | Использование как сырьё/материал |
| **Возврат поставщику** | Возврат ненужного товара |
| **Прочее** | Другие случаи |

## Экспорт для отчёта

Для удобства подачи отчёта используйте экспорт выбытия:

1. Перейдите в **«Настройка»**
2. Нажмите **«Экспорт выбытия (CSV)»**
3. В файле будут все единицы с заполненными данными выбытия

## Отслеживание

Во вкладке **«Вывод из оборота»** вы видите:
- Единицы, требующие отчёта
- Статус каждого отчёта
- Фильтр по статусу и дате

## Важно

- Отчёт о выбытии обязателен для товаров с маркировкой
- Без подачи отчёта товар остаётся в обороте в системе ЧЗ
- Дедлайн: **3 рабочих дня** с момента выбытия
- Для экспорта за пределы ЕАЭС используйте причину «Экспорт за пределы ЕАЭС»
