# 13. Возврат товара

## Когда происходит возврат

- Покупатель вернул товар в магазин
- Маркетплейс вернул товар продавцу
- Товар оказался бракованным

## Процесс возврата

### Шаг 1: Оформите возврат в приложении

1. Перейдите во вкладку **«Продано»**
2. Найдите проданный товар
3. Нажмите **«Возврат»**
4. Выберите **склад**, на который возвращается товар
5. Подтвердите

### Что происходит автоматически

- Статус товара меняется на **«В обороте»** (INTRODUCED)
- Товар перемещается на указанный склад
- Очищаются данные продажи (номер заказа, дата продажи)
- Устанавливается флаг **«Был возврат»**

### Шаг 2: Подайте отчёт о возврате через ЛК ЧЗ

**Обязательно!** После возврата товара необходимо подать отчёт:

1. Войдите в личный кабинет Честный Знак: https://markirovka.crpt.ru
2. Перейдите в раздел **«Отчёты»** → **«Отчёт о возврате»**
3. Создайте новый отчёт
4. Заполните:
   - **ИНН** организации
   - **Код продукции** (GTIN)
   - **Серийный номер** — из КМ кода
   - **Причина возврата**: «Возврат при дистанционном способе продажи»
   - **Дата возврата**
   - **Документ-основание**: акт приёма-передачи, чек возврата
5. Подпишите отчёт ЭЦП
6. Отправьте

### Шаг 3: Обновите статус

1. Подождите 1-2 дня
2. Нажмите **«Обновить статусы всех товаров»**
3. Статус товара подтвердится

## Важные замечания

### Товар с флагом «Был возврат»

- Отображается с красным выделением в таблице
- Это предупреждение: товар был возвращён и требует отчёта

### Ограничение

- Возврат доступен только для товаров со статусом **«В обороте»** (INTRODUCED)
- Если статус RETIRED (Выбыл) — возврат невозможен через интерфейс
- Для таких случаев обратитесь в поддержку ЧЗ

### Дублирование маркировки

При возврате товара КМ код остаётся тем же. Дубликат этикетки не требуется — товар уже промаркирован.

## Схема возврата

```
Товар продан (RETIRED)
    ↓
Покупатель возвращает товар
    ↓
Оформление возврата в приложении
    ↓
Товар возвращён на склад (INTRODUCED)
    ↓
Подача отчёта о возврате через ЛК ЧЗ
    ↓
Товар снова доступен для продажи
```
