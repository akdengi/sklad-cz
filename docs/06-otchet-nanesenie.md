# 6. Подача отчёта о нанесении

## Что это такое

После нанесения кодов маркировки на товар необходимо подать отчёт о нанесении в систему Честный Знак. Это обязательный шаг перед вводом товара в оборот.

## Порядок действий

### Шаг 1: Нанесите коды на товар

Убедитесь, что все коды маркировки нанесены на физические товары:
- Отсканируйте коды через вкладку **«Сканирование»**
- Или импортируйте из CSV/TXT через вкладку **«Импорт»**
- Статус единиц должен быть **«Нанесён»**

### Шаг 2: Подайте отчёт через ЛК ЧЗ

1. Войдите в личный кабинет Честный Знак: https://markirovka.crpt.ru
2. Перейдите в раздел **«Отчёты»** → **«Отчёт о нанесении»**
3. Создайте новый отчёт
4. Заполните данные:
   - **ИНН** организации-производителя или импортёра
   - **Код продукции** (GTIN)
   - **Список серийных номеров** — серийные номера из ваших КМ
5. Подпишите отчёт ЭЦП
6. Отправьте отчёт

### Шаг 3: Обновите статусы в приложении

После обработки отчёта ЧЗ:
1. Перейдите во вкладку **«Настройка»**
2. Нажмите **«Обновить статусы всех товаров»**
3. Система запросит актуальные статусы из API ЧЗ
4. Статусы единиц обновятся автоматически

## Формат серийных номеров для отчёта

Из КМ кода извлекается серийный номер — часть после AI 21:

```
КМ:       010460000000000021АБВГДЕЖЗИЙ
Серийный: АБВГДЕЖЗИЙ
```

## Экспорт для отчёта

Для удобства подачи отчёта используйте экспорт выбытия:
1. Перейдите в **«Настройка»**
2. Нажмите **«Экспорт выбытия (CSV)»**
3. В файле будут три колонки:
   - Полный КМ код
   - Код для отчёта (до GS+91)
   - Формат для Озон/маркетплейса
