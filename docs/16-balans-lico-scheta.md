# Баланс лицевого счёта

## Что это

На дашборде (вкладка «Сводка») отображается баланс денежных средств оператора маркировки (УОТ) по выбранной товарной группе.

## Как работает

1. При загрузке дашборда приложение автоматически запрашивает баланс через API Честный Знак
2. Баланс отображается в карточке «Баланс лицевого счёта»
3. Нажмите **«Обновить баланс»** для ручного обновления

## Отображаемая информация

| Поле | Описание |
|------|----------|
| **Баланс (руб.)** | Текущий баланс лицевого счёта в рублях (переведено из копеек) |
| **Договор** | Идентификатор договора УОТ с ГИС МТ |
| **Организация** | ID организации УОТ в ГИС МТ |
| **Название товарной группы** | Название выбранной товарной группы из настроек |

## API-эндпоинт

- **URL**: `GET /api/balance`
- **Ответ**: JSON с полями `ok`, `balance`, `contract_id`, `organisation_id`, `product_group_id`, `product_group_name`
- **Ошибки**: `ok: false` + `error` с описанием

## Требования

- Настроенный сертификат ЭЦП (КриптоПро CSP)
- Авторизация в API Честный Знак (токен получается автоматически)
- Выбранная товарная группа в настройках
