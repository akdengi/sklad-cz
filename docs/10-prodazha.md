# 10. Процесс продажи товара

## Общая схема

```
Товар на складе
    ↓
Продажа через кассу / маркетплейс
    ↓
Оформление продажи в приложении
    ↓
(Для ЧЗ) Подача отчёта о выводе из оборота через ЛК ЧЗ
```

## Продажа товаров с маркировкой (ЧЗ)

### Продажа через кассу (офлайн)

1. Покупатель приходит с товаром
2. Кассир сканирует EAN-13 штрихкод на кассе
3. Пробивается чек
4. В приложении: **«Продажа»** → сканируйте **КМ код** → выберите склад → введите номер заказа и цену → **«Продать»**
5. Подайте отчёт о выбытии через ЛК ЧЗ

### Продажа через маркетплейс (на примере Озон)

1. Получите **номер заказа Озон** (из кабинета продавца)
2. **«Продажа»** → сканируйте КМ → выберите **виртуальный склад «Озон»** → введите номер заказа и цену → **«Продать»**
3. Озон автоматически подаёт отчёты о выбытии для маркированных товаров

### Что происходит автоматически

- Товар перемещается с физического склада на виртуальный маркетплейса
- Статус меняется на **«Продан»**
- Заполняются данные для отчёта о выбытии (тип, причина, адрес, FIAS)

### Дедлайны

После продажи необходимо подать отчёт о выбытии в течение **3 рабочих дней**. На странице **«Сводка»** отображаются предупреждения:
- **Красный** — дедлайн просрочен
- **Жёлтый** — дедлайн через 1-2 дня

## Продажа товаров без маркировки

### Особенности

- Продаётся по **EAN-13** штрихкоду или артикулу
- **Без** отчёта о выбытии в ЧЗ
- Учёт только: дата продажи, номер заказа/чека, сумма
- Возврат на склад без ограничений

### Процесс

1. Перейдите во вкладку **«Продажа»**
2. Отсканируйте **EAN-13** штрихкод товара (или вставьте вручную)
3. Система найдёт товар без маркировки
4. Выберите **склад назначения** (виртуальный маркетплейса)
5. Введите **номер заказа/чека**
6. Введите **цену за единицу**
7. Нажмите **«Продать»**

### Что происходит

- Товар перемещается на виртуальный склад
- Статус меняется на **«Продан»**
- **Без** заполнения полей выбытия (адрес, FIAS, тип операции)
- **Без** необходимости подавать отчёт в ЧЗ

## Быстрая продажа (Quick Sell)

Самый быстрый способ для обоих типов товаров:

1. Вкладка **«Продажа»**
2. Сканируйте **КМ** (для ЧЗ) или **EAN-13** (для без ЧЗ)
3. Система автоматически определит тип товара
4. Выберите склад назначения
5. Введите номер заказа и цену
6. **«Продать»** — готово
